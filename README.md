# Jemilio map 1 : Mable island!
A map for cities skylines!

# how to install?
 Put jemiliomap1.crp in C:\Users\YOUR_ACCOUNTNAME\AppData\Local\Colossal Order\Cities_Skylines\Maps.
 Restart/start your game.
 Hit new game.
 Press Mable Island.
 Start the game!
